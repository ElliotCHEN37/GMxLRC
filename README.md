# GMxLRC
<img src="icon.png" height="128"><br>
A simple GUI for MxLRC<br>
Based on v1.2.2

## Additional Feature
- Auto-read token from file
1. Create `token.txt` in the working dir
2. Paste your user token into `token.txt`

## Requirements
- Python 3.9+
- pyqt5

## Build
1. clone this repo
2. cd to folder
3. run `pyinstaller build.spec` in terminal
